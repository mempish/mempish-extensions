# Template Extensions

Paperback extensions designed as templates to help developers get started quickly.

This includes the following:

- Package config
- TypeScript config
- ESLint config
- Prettier config
- Dependabot config
- Development setup
- Husky pre-push hook
- GitHub CI/CD Actions
- VSCode config files (settings + extensions)
- A basic example content extension
- A basic example tracker extension

All extension repositories in the Inkdex GitHub Organization use this one as their base template.

## Available Extensions

- Bato.To
- VyVyManga (Upcoming)

## Installation

To add these extensions to Paperback, it’s recommended to install just this repository by visiting [this webpage][repository-installation-page].
